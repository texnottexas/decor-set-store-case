The scorecard line nobody is watching

Our store conversion and engaged-spender curves are quietly capped by one setting that has not been revisited in roughly a year: older decoration sets stopped graduating into the Decor Set Token store. This doc reframes that frozen setting as an unowned metric, and proposes the lowest-risk lever we own to lift it.

The blind spot

A config nobody revisits is capping a line we are graded on

Store conversion and engaged-spender retention both depend on players having a next reachable goal. When older sets stopped graduating, that goal quietly disappeared, and the line flattened on its own. No dashboard alerts on a setting that simply never changes, so the cap looks like a baseline.

Patient players now bank free tokens with nothing to claim. Paying players, with no reachable target, stop topping up. The 10-set backlog earns nothing while it sits.

The lever

Tie graduation to the cadence we already run

Every time a brand-new set launches, about one every 2.5 months, the oldest shelved set graduates into the token store. One launches, one graduates. Premium launches stay premium and the monthly pop-up pack re-releases stay intact.

2.7x the backlog token revenue versus leaving the cadence frozen, on the illustrative model below.
02

Hypothesis

IF / THEN

If the oldest shelved set graduates into the Decor Set Token store on each new-set launch (about 5 per year), then store conversion and weekly top-up rate among engaged spenders recover, because every player again has a reachable goal to spend toward, without reducing premium launch-pack sales, since the graduating sets earn nothing under the frozen setting today.

Experiment design and rollout

A/B: frozen versus steady

Control / A
Frozen cadence
Backlog stays shelved, current setting. Measures the silent cap as it stands.
Treatment / B
Steady graduation
Oldest shelved set graduates on each new-set launch. One in, one graduates.
  • UnitServer cohort, or staged servers if a clean split is easier than a per-player split.
  • PrimaryBacklog store conversion rate and weekly top-up rate among engaged spenders.
  • GuardrailPremium launch-pack revenue for the newest 5 sets must hold flat.
  • Window2 to 3 graduation cycles, roughly one to two quarters.

Rollout plan

  1. Flip the cadence setting
    Queue the 10 older sets oldest-first behind the existing new-set launch trigger.
  2. Graduate one on next launch
    The oldest shelved set enters the token store the day the next new set ships.
  3. Watch the guardrail
    Confirm premium launch-pack revenue holds while conversion lifts.
  4. Continue oldest-first
    Repeat each launch until the 10-set backlog is cleared, then the lever rests.
08

Risks addressed

"This cannibalizes premium launch-pack sales."
The 10 graduating sets are the older backlog whose premium windows closed years ago. They earn $0 today, so there is no premium revenue to cannibalize. The newest 5 sets stay premium and untouched, and a reachable token-store goal is what keeps players topping up at all. This grows revenue rather than trading it.
Mitigated
"It dilutes the pop-up packs."
The monthly pop-up pack re-releases stay exactly as they are. Only the 10 older sets ever graduate, and only into the token store. The two systems run side by side.
No change
"The backlog could run out and over-supply the store."
It is finite by design. After 10 graduations the lever simply rests. Cadence is tied to launches we already run, so the store never outpaces new content.
Self-limiting
As a gentle side benefit, completed sets grant a small permanent march-size bonus, so steady graduation also narrows the army-size gap a little for players who spend less. This is a neutral fairness note, not a request, and it is fully optional to the revenue case above.
Bonus

The highest-leverage, lowest-risk lever on this surface

No new art, no new code, no new system. One store-config setting moved from frozen to a cadence we already run. It lifts store conversion and engaged-spender retention, clears a backlog that earns nothing today, and on the illustrative model leaves roughly $47,676 a year on the table for as long as it stays frozen. This is an easy win to capture whenever the team is ready.
