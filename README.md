# Decor Set Store — A Case for Steady Releases

An interactive, data-grounded look at Top War's **Decor Set Store**, and why letting the
older decoration-set backlog graduate into the store on a steady cadence earns more than
holding it back — while leaving the newest sets and the monthly pop-up packs untouched.

**Live page → https://texnottexas.github.io/decor-set-store-case/**

## What's inside
- The full 23-set catalogue (8 in the token store · 10 older sets shelved · 5 recent premium)
- The token economy and the deliberate weekly-drip pricing
- Two interactive models: the "wall" a caught-up player hits, and steady-release vs hold-and-dump revenue

## Notes on the numbers
Set names, prices, release dates, and the token economy are read from the live game client.
The revenue figures are an **illustrative** model — its assumptions (player counts, tier mix,
release cadence) are exposed as on-page controls so they can be adjusted and challenged.

Single self-contained HTML file. No build step, no dependencies, no tracking.
